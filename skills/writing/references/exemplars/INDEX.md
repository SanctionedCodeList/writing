## Brief Writing Exemplars

### Top Litigator Techniques

Extracted from briefs by Roberts, Clement, Olson, Boies, and other elite advocates:

1. **Storytelling discipline** — Facts as narrative, not recitation
2. **Showing vs. telling** — Let actions demonstrate; avoid adjectives
3. **Varied transitions** — Not "Moreover/Furthermore" chains
4. **Sentence rhythm** — Strategic length variation
5. **Parallelism** — Balanced structure for emphasis
6. **Drawing lines** — Clear standards, not mush
7. **Antithesis** — Contrasting structures

## Executive Document Exemplars

### Consulting-Style Documents

Key patterns from McKinsey, BCG, Bain style:

1. **Ghost structure first** — Write all headings before content
2. **Storyline test** — Headings alone tell complete story
3. **One big number per page** — Make key metrics unmissable
4. **SCQA opening** — Situation → Complication → Question → Answer
